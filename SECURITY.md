# Security Policy

## Supported Versions

Security fixes are handled on the latest `main` branch and the most recent npm
package version.

## Reporting a Vulnerability

Please report suspected vulnerabilities by opening a private GitHub security
advisory for this repository. Include:

- the affected version or commit
- a minimal fixture or command that reproduces the issue
- expected and observed behavior
- any known workaround

Do not include secrets, private repository evidence, customer data, or raw
incident material in public issues.

## Scope

This package is a local CLI and reusable skill. It bundles supplied repository
evidence, command notes, and claims into local review artifacts; it should not
publish, upload, or contact external services as part of normal operation.
